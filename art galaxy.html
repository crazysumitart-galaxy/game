  • INTRODUCTION
    • Tic Tac Toe is a popular two-player logic-based game.
    • In this project, I have developed a digital version of the Tic Tac Toe game.
    • The game provides two playing options:
      • Player vs Player Player vs Computer
    • Users can enter their names, and the game runs smoothly in a web browser.
  • TECHNOLOGIES USED
    1. HTML (Structure)
      2. HTML is used to create the basic skeleton of the game.
    2. CSS (Design)
      3. CSS is used to make the game colorful, attractive, and visually appealing.
    3. JavaScript (Logic)
      4. JavaScript acts as the brain of the game and handles:
    1. Turn changing
    2. Win checking
    3. Computer moves
    4. Game restart




      5. Enter names & select mode



      • FEATURES
        1. User-friendly interface
        2. Player vs Player mode
        3. Player vs Computer mode
        4. Dynamic game messages
        5. Restart option
